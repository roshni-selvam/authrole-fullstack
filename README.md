# 🔐 Role-Based Authentication System

A full-stack authentication app with USER and ADMIN role management.

## Features
- 👤 User & Admin separate dashboards
- 🔒 BCrypt password encryption
- 🛡️ Spring Security 6 integration
- 🌐 CORS-configured REST APIs
- 💾 MySQL database

## Tech Stack
- React + Vite (Frontend)
- Spring Boot + Spring Security 6 (Backend)
- MySQL + BCrypt
